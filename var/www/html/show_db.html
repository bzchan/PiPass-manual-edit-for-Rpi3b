Database Sources

By default, PiPass uses its own Google Spreadsheet for data parsing and consistency reasons. Refer to the README on GitHub to learn more about customization.

The default worksheet suggested is the combined Nintendo Zone and Extended MAC list (Worksheet #7) because it will be good enough for most people who just want to StreetPass.

All of the Nintendo Zone information has been taken from this spreadsheet. It's maintained by FatMagic, so credits to FatMagic for putting the work into maintaining it.
